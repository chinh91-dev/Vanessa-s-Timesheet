# Comans Services Timesheet & Reporting App

A full-stack TypeScript application for recording time, managing projects, and exporting rich reports – built for Comans Services staff and admins.

---

## Editing the code

### 1 · Work inside **Lovable**

1. Open the project link above.  
2. Prompt in the sidebar or edit any file – Lovable will auto-commit to `main`.

### 2 · Work locally in your own IDE

```bash
# Clone the repo
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>

# Install dependencies (Node.js + npm required – nvm is easiest)
npm install

# Start the dev server with fast refresh
npm run dev
